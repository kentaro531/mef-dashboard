# 数学教育フェス KPIダッシュボード

## セットアップ手順

### ① Googleスプレッドシートの準備（5分）

1. `数フェスKPI管理シート.xlsx` をGoogle Driveにアップロード
2. 右クリック →「アプリで開く」→「Googleスプレッドシート」
3. **「ファイル」→「共有」→「ウェブに公開」**
   - 「ドキュメント全体」を選択
   - 形式は「カンマ区切り形式（.csv）」
   - 「公開」をクリック
4. スプレッドシートのURLをコピーしておく
   - 例: `https://docs.google.com/spreadsheets/d/1ABC.../edit`

### ② ダッシュボードのデプロイ（5分）

**前提: Node.js 18以上がインストールされていること**

```bash
# 依存パッケージのインストール
npm install

# ローカルで動作確認
npm run dev
# → http://localhost:5173 で確認

# ビルド
npm run build
```

**Vercelへのデプロイ:**

1. [vercel.com](https://vercel.com) にGitHubアカウントでログイン
2. このフォルダをGitHubリポジトリにpush
3. Vercelで「Import Project」→ リポジトリを選択
4. フレームワーク: `Vite` を選択（自動検出される）
5. 「Deploy」をクリック
6. デプロイ完了後、URLが発行される（例: `mef-dashboard.vercel.app`）

**または、Vercel CLIで:**

```bash
npx vercel
```

### ③ ダッシュボードとシートの連携（1分）

1. デプロイしたダッシュボードを開く
2. 右上の ⚙️ ボタンをクリック
3. ①でコピーしたスプレッドシートのURLを貼り付け
4. 「接続」をクリック
5. 「Sheets連携中」と表示されれば成功

## 運用方法

### チームメンバーの作業

**行動KPI（毎週更新）:**
- Googleスプレッドシートの「行動KPI」シートを開く
- **D列（黄色セル）** に実績値を記入する
  - 例: EDIX東京に行ったら → `1` に変更
  - 例: YouTubeショートを5本投稿したら → `5` に変更
- H列に記入者名を入れる

**結果KPI（毎月更新）:**
- 「結果KPI」シートの **B列（黄色セル）** を更新
  - 有効メンバー数、月次売上 など

**月次推移（毎月末）:**
- 「月次推移」シートに1行追加
  - 年月、新規登録数、退会数、純増減、累計有効

### ダッシュボード側

- シートを更新すれば、ダッシュボードに自動で反映
- 手動更新したい場合は ⚙️ → 🔄 ボタン

## ファイル構成

```
mef-dashboard/
├── index.html          # エントリーポイント
├── package.json        # 依存パッケージ
├── vite.config.js      # Vite設定
└── src/
    ├── main.jsx        # Reactマウント
    └── App.jsx         # ダッシュボード本体
```

## シート構成

| シート名 | 用途 | 更新頻度 |
|---------|------|---------|
| 行動KPI | やったかどうかの記録 | 毎週 |
| 結果KPI | 成果指標の記録 | 毎月 |
| 月次推移 | 会員数の推移 | 毎月末 |
